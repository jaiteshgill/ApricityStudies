UNIT 3: Quadratic Functions

Welcome to Unit 3: Quadratic Functions! In this section, we'll dive deep into the understanding of quadratic functions, exploring everything from vertex form to standard form, how to graph them, and how to interpret their behavior. Quadratic functions play a significant role in mathematics due to their unique U-shaped graph, known as a parabola. These functions are not only a fundamental part of algebra but also have real-world applications, making them essential for problem-solving.

General Form of a Quadratic Equation is:
ax² + bx + c = 0


Quadratic Functions in Vertex Form

A quadratic function in vertex form can be expressed as: f(x) = a(x - p)2 + q. In this form:

  • a determines the direction of the parabola and whether it is wide or narrow.
  • (p, q) is the vertex, which tells us the highest or lowest point of the graph.
  • The axis of symmetry is the vertical line x = p.
  • The parabola is congruent to f(x) = ax², translated horizontally by p units and vertically by q units.

Remember: If a > 0, the parabola opens upward, making the vertex the lowest point. If a < 0, the parabola opens downward, making the vertex the highest point.

Transforming f(x) = x² into Vertex Form

To graph a quadratic function in vertex form, we start by transforming the graph of f(x) = x². This transformation involves shifting the graph horizontally and vertically based on the values of p and q. Let’s break this down:

  • If a > 0, the parabola opens upward, and if a < 0, it opens downward.
  • If -1 < a < 1, the parabola is wider compared to the standard graph of f(x) = x².
  • If a > 1 or a < -1, the parabola becomes narrower.

The parameter q determines the vertical position of the parabola. A positive value for q shifts the graph upward, while a negative value shifts it downward. Similarly, the parameter p dictates the horizontal shift: a positive value moves the graph to the right, and a negative value moves it to the left.

Determining the Number of X-Intercepts

One of the most important aspects of graphing a quadratic function is understanding how many times it crosses the x-axis, known as the x-intercepts. This can be determined by analyzing the values of a and q:

  • If a > 0 and q > 0, the graph opens upward and the vertex is above the x-axis, meaning there are no x-intercepts.
  • If a > 0 and q = 0, the vertex is on the x-axis, and there is exactly one x-intercept.
  • If a > 0 and q < 0, the graph crosses the x-axis twice, resulting in two x-intercepts.

Quadratic Functions in Standard Form

Quadratic functions can also be expressed in standard form, which is given by f(x) = ax² + bx + c, where a is non-zero. In this form:

  • The graph is a parabola that is symmetric about a vertical line called the axis of symmetry.
  • The y-intercept of the graph is the point (0, c).

The x-coordinate of the vertex is:
x = -b / 2a

The standard form is particularly useful for determining important features of the parabola such as the direction of opening, vertex, axis of symmetry, and the intercepts. By using these characteristics, we can sketch the graph of the quadratic function.

Application of Quadratic Functions in Real Life

Quadratic functions are often used to model real-world situations. Some examples include:

  • The path of a ball thrown in the air can be modeled using a quadratic function.
  • The profit of a business based on the number of items sold can be represented by a quadratic function.
  • Quadratics are also used in architecture and engineering to design structures with parabolic arches.

Converting Standard Form to Vertex Form: Completing the Square

To convert a quadratic function from standard form to vertex form, we use a method called completing the square. This process involves manipulating the equation so that we can rewrite it in the form f(x) = a(x - p)² + q.

Let’s look at an example:

y = 5x² - 30x + 7

Follow these steps to complete the square:

  1. Factor out the coefficient of x² from the first two terms:
    y = 5(x² - 6x) + 7
  2. Add and subtract a number to make the quadratic expression a perfect square trinomial:
    y = 5(x² - 6x + 9 - 9) + 7
  3. Rewrite the trinomial as a square:
    y = 5[(x - 3)² - 9] + 7
  4. Simplify the expression:
    y = 5(x - 3)² - 45 + 7
  5. Final vertex form:
    y = 5(x - 3)² - 38

Converting the quadratic function to vertex form reveals that the vertex of the parabola is at (3, -38), making it easier to understand the graph's behavior.
